Última actualización: 1 de diciembre de 2025

Introducción

ControlD Remote ("nosotros", "nuestro" o "la aplicación") es una aplicación iOS diseñada para gestionar tu servicio DNS de ControlD directamente desde tu dispositivo. Esta Política de Privacidad explica qué datos recopilamos, cómo los utilizamos y tus derechos con respecto a tu información.

Información de Contacto:

Email: punterodev@gmail.com

1. Información que Recopilamos

1.1 Información que Proporcionas

Clave API de ControlD

  • Qué: Tu clave API personal de ControlD necesaria para autenticarse con el servicio de ControlD
  • Almacenamiento: Cifrada en el Keychain de iOS con protección kSecAttrAccessibleWhenUnlockedThisDeviceOnly
  • Sincronización: NO sincronizada con iCloud Keychain
  • Propósito: Necesaria para comunicarse con la API de ControlD en tu nombre

1.2 Información Recopilada Automáticamente

Preferencias Locales (UserDefaults)

  • Personalización del orden de pestañas
  • Preferencias de formato de fecha (relativo/absoluto)
  • Configuración de tema de la interfaz
  • Preferencias de visualización (visibilidad de latencia, configuración de notificaciones)
  • ID de perfil predeterminado
  • Búsquedas recientes de dominios
  • Marcas de tiempo de última actualización
  • Estado de inicio de sesión (booleano)

Propósito: Todas las preferencias se almacenan localmente para personalizar tu experiencia. Ninguno de estos datos se transmite a ningún servidor.

Base de Datos Local (SwiftData)

La aplicación mantiene una base de datos local en tu dispositivo que contiene:

Tipo de Dato Propósito Almacenado Localmente
Dispositivos Caché de tus dispositivos ControlD ✅ Sí
Perfiles Perfiles DNS y sus configuraciones ✅ Sí
Cuenta de Usuario Detalles de tu cuenta de ControlD ✅ Sí
Suscripciones Información de tu suscripción a ControlD ✅ Sí
Historial de Pagos Registros de pagos anteriores de ControlD ✅ Sí
Registros DNS Registros históricos de consultas DNS ✅ Sí
Iconos de Servicios Iconos de servicios en caché (archivos SVG) ✅ Sí
Marcas de Tiempo Marcas de tiempo de invalidación de caché ✅ Sí
  • Ubicación de almacenamiento: ~/Library/Application Support/ (almacenamiento predeterminado de iOS)
  • Cifrado: Protegido por cifrado a nivel de dispositivo de iOS
  • Sincronización iCloud: NO habilitada - todos los datos permanecen en tu dispositivo

1.3 Datos de Actividad de Red

La aplicación se conecta a los servidores de ControlD para proporcionar su funcionalidad:

Conexiones API:

  • https://api.controld.com - API principal para gestión de perfiles, dispositivos, filtros, reglas y servicios
  • https://europe.analytics.controld.com - Recuperación de registros DNS y analíticas

Datos Transmitidos:

  • Tu clave API (mediante autenticación Bearer token en encabezados HTTPS)
  • Cambios de configuración DNS que realizas a través de la aplicación
  • Solicitudes de perfiles, dispositivos, reglas, filtros y registros DNS
  • Operaciones de gestión de dispositivos (crear, actualizar, eliminar)

Importante: No interceptamos, almacenamos ni analizamos este tráfico de red. Todos los datos van directamente entre tu dispositivo y los servidores de ControlD mediante conexiones HTTPS cifradas.

1.4 Datos Técnicos (APIs de Razón Requerida)

En cumplimiento con los requisitos de la App Store de Apple, utilizamos las siguientes APIs del sistema:

Tipo de API Código de Razón Propósito

Nota: Estas APIs acceden solo a metadatos técnicos (marcas de tiempo, tiempo de actividad del sistema). No ocurre recopilación de datos personales ni seguimiento.

3. Almacenamiento y Seguridad de Datos

3.1 Almacenamiento Local

Todos los datos se almacenan localmente en tu dispositivo:

  • Keychain: Clave API (cifrada por hardware, accesible solo cuando el dispositivo está desbloqueado)
  • UserDefaults: Preferencias (en sandbox de esta aplicación)
  • SwiftData: Caché y datos históricos (cifrado del sistema de archivos de iOS)

Medidas de Seguridad:

  • Acceso thread-safe al Keychain mediante DispatchQueue
  • Sin transmisión de datos a servidores excepto para llamadas API a ControlD
  • Todas las llamadas de red usan HTTPS (TLS 1.2+)
  • Sin sincronización con iCloud (previene exposición de datos)

3.2 Servicios de Terceros

ControlD (api.controld.com, europe.analytics.controld.com)

  • Propósito: Servicio principal de gestión DNS
  • Datos Compartidos: Clave API, configuraciones DNS, información de dispositivos
  • Su Política de Privacidad: https://controld.com/privacy
  • Sus Términos de Servicio: https://controld.com/terms

Apple StoreKit

  • Propósito: Procesar donaciones voluntarias
  • Datos Compartidos: ID de transacción, ID de producto, fecha de compra (gestionado por Apple)
  • Privacidad: Regida por la Política de Privacidad de Apple
  • Nota: Las donaciones son productos consumibles; sin suscripciones recurrentes

3.3 Dependencias de Código Abierto

SwipeActions (https://github.com/aheze/SwipeActions)

  • Tipo: Biblioteca de gestos UI
  • Licencia: MIT (código abierto)
  • Privacidad: Sin recopilación de datos, funcionalidad puramente UI
  • Auditoría de Código: Disponible públicamente en GitHub

4. Retención y Eliminación de Datos

4.1 Cuánto Tiempo Conservamos los Datos

Tipo de Dato Período de Retención Método de Eliminación
Clave API (Keychain) Hasta que cierres sesión o elimines la app Automático al cerrar sesión/eliminar app
UserDefaults Hasta que elimines la app Automático al eliminar la app
Caché SwiftData Hasta que limpies caché o elimines la app Limpieza manual en Ajustes o eliminación de app
Registros DNS Almacenados localmente hasta limpieza manual Eliminación iniciada por el usuario

Datos del Lado del Servidor: No controlamos los datos almacenados en los servidores de ControlD. Para eliminar los datos de tu cuenta de ControlD, visita su panel de control o contacta con el soporte de ControlD.

4.2 Cómo Eliminar tus Datos

Opción 1: Cerrar Sesión

  1. Abre ControlD Remote
  2. Ve a la pestaña Ajustes
  3. Toca Cerrar Sesión
  4. Esto elimina tu clave API del Keychain

Opción 2: Eliminar la Aplicación

  1. Mantén presionado el icono de la app
  2. Selecciona Eliminar App
  3. Confirma la eliminación
  4. Esto elimina todos los datos locales inmediatamente

Importante: Eliminar la aplicación NO elimina tu cuenta de ControlD. Para eliminar tu cuenta de ControlD, inicia sesión en https://controld.com/dashboard y solicita la eliminación de cuenta.

6. Privacidad de Menores

Restricción de Edad: ControlD Remote no está destinado a menores de 13 años.

No recopilamos información personal de menores de 13 años de manera intencionada. Si crees que un menor nos ha proporcionado información personal, contáctanos inmediatamente y la eliminaremos.

Consentimiento Parental: Si tienes entre 13 y 18 años, obtén el consentimiento de tus padres antes de usar esta aplicación.
