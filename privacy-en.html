Introduction

ControlD Remote ("we," "our," or "the app") is an iOS application designed to manage your ControlD DNS service directly from your device. This Privacy Policy explains what data we collect, how we use it, and your rights regarding your information.

Contact Information:

Email: punterodev@gmail.com

1. Information We Collect

1.1 Information You Provide

ControlD API Key

  • What: Your personal ControlD API key required to authenticate with the ControlD service
  • Storage: Encrypted in iOS Keychain with kSecAttrAccessibleWhenUnlockedThisDeviceOnly protection
  • Sync: NOT synchronized with iCloud Keychain
  • Purpose: Required to communicate with ControlD's API on your behalf

1.2 Information Automatically Collected

Local Preferences (UserDefaults)

  • Tab order customization
  • Date format preferences (relative/absolute)
  • UI theme settings
  • Display preferences (latency visibility, notification settings)
  • Default profile ID
  • Recent domain searches
  • Last refresh timestamps
  • Login state (boolean)

Purpose: All preferences are stored locally to personalize your experience. None of this data is transmitted to any server.

Local Database (SwiftData)

The app maintains a local database on your device containing:

Data Type Purpose Stored Locally
Devices Cache of your ControlD devices ✅ Yes
Profiles DNS profiles and their configurations ✅ Yes
User Account Your account details from ControlD ✅ Yes
Subscriptions Your ControlD subscription information ✅ Yes
Payment History Past payment records from ControlD ✅ Yes
DNS Logs Historical DNS query logs ✅ Yes
Service Icons Cached service icons (SVG files) ✅ Yes
Refresh Timestamps Cache invalidation timestamps ✅ Yes
  • Storage Location: ~/Library/Application Support/ (iOS default app storage)
  • Encryption: Protected by iOS device-level encryption
  • iCloud Sync: NOT enabled - all data remains on your device

1.3 Network Activity Data

The app connects to ControlD's servers to provide its functionality:

API Connections:

  • https://api.controld.com - Primary API for profile, device, filter, rule, and service management
  • https://europe.analytics.controld.com - DNS logs and analytics retrieval

Data Transmitted:

  • Your API key (via Bearer token authentication in HTTPS headers)
  • DNS configuration changes you make through the app
  • Requests for profiles, devices, rules, filters, and DNS logs
  • Device management operations (create, update, delete)

Important: We do not intercept, store, or analyze this network traffic. All data goes directly between your device and ControlD's servers via encrypted HTTPS connections.

1.4 Technical Data (Required Reason APIs)

In compliance with Apple's App Store requirements, we use the following system APIs:

API Type Reason Code Purpose
UserDefaults CA92.1 Store user preferences for app functionality
File Timestamps C617.1, DDA9.1 Determine cache validity (C617.1) and debug logging in development builds only (DDA9.1)
System Boot Time 35F9.1 Measure network connectivity for the Network framework's connection monitoring

Note: These APIs access only technical metadata (timestamps, system uptime). No personal data or tracking occurs.

3. Data Storage and Security

3.1 Local Storage

All data is stored locally on your device:

  • Keychain: API Key (hardware-encrypted, accessible only when device unlocked)
  • UserDefaults: Preferences (sandboxed to this app)
  • SwiftData: Cache and historical data (iOS file system encryption)

Security Measures:

  • Thread-safe Keychain access via DispatchQueue
  • No data transmission to servers except for API calls to ControlD
  • All network calls use HTTPS (TLS 1.2+)
  • No iCloud synchronization (prevents data exposure)

3.2 Third-Party Services

ControlD (api.controld.com, europe.analytics.controld.com)

  • Purpose: Core DNS management service
  • Data Shared: API Key, DNS configurations, device information
  • Their Privacy Policy: https://controld.com/privacy
  • Their Terms of Service: https://controld.com/terms

Apple StoreKit

  • Privacy: Governed by Apple's Privacy Policy
  • Note: Donations are consumable products; no recurring subscriptions

4. Data Retention and Deletion

4.1 How Long We Keep Data

UserDefaults Until you delete the app Automatic on app deletion
SwiftData Cache Until you clear cache or delete the app Manual clear in Settings or app deletion
DNS Logs Cached locally until manually cleared User-initiated deletion

ControlD Support: help@controld.com

4.2 How to Delete Your Data

Option 1: Log Out

  1. Open ControlD Remote
  2. Go to Settings tab
  3. Tap Log Out
  4. This deletes your API key from Keychain

Option 2: Delete the App

  1. Long-press the app icon
  2. Select Delete App
  3. Confirm deletion
  4. This removes all local data immediately

Important: Deleting the app does NOT delete your ControlD account. To delete your ControlD account, log in to https://controld.com/dashboard and request account deletion.
